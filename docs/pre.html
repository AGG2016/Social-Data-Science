The predict() function

predict(lm,newdata)

  • “newdata” must be a dataframe with a variable with the same name as the explanatory variable.
  • The result is a vector of fitted values.

vm<-broom::augment(lm,newdata=new_data)

Adding a Regression Line Manually

coefs<-coef(mod)
coefs<-as.data.frame(t(as.data.frame(coefs)))
ggplot(data = murders, aes(x = GunOwnership, y = GunMurders)) + 
  geom_point(col="dodgerblue",alpha=0.9) + 
  geom_abline(data = coefs, 
              aes(intercept = `(Intercept)`, slope = GunOwnership),
              col="dodgerblue")
