Measurements of Variability

Range

\[Max_{ob}-Min_{ob}\]

Problem: ignores \(n-2\) observations.

Sum of Deviations from Mean

Is it possible to simply add deviations to have a global indicator of how deviated -overall- are the observations from their central tendency?

¡No! ¡Because some are negative and others positive, they will always sum zero!

Don’t you believe?

Demonstration

\[\sum(x-\bar{x})=\sum x_i-\sum \bar{x}=\sum x_i-n\bar{x}=\sum x_i-n(\frac{1}{n}\sum x_i)=0\]

Sample Variance

\[s^2=\frac{\sum(x_i-\bar{x})^2}{n-1}\]

Motivation for \(n-1\):

\(\mu\) is -in general- unknown.

Problem: does not preserve the units of the phenomenon.

Standard Deviation

\[s=\sqrt{s^2}\]

Advantage: same units of the sample.

Example

library(classInt)
intervals=classIntervals(murders$Population,4,style="quantile",
                         intervalClosure = "left")
PopulationGroup=cut(murders$Population,breaks = intervals$brks,
                    labels=c("Small","Average -","Average +","Big"))
murders<-cbind(murders,PopulationGroup)
murders[51,9]<-"Small"
big<-subset(murders,PopulationGroup=="Big")
rbig<-max(big$GunMurders)-min(big$GunMurders)
sbig<-sd(big$GunMurders)
small<-subset(murders,PopulationGroup=="Small")
rsmall<-max(small$GunMurders)-min(small$GunMurders)
ssmall<-sd(small$GunMurders)
ggplot(data=murders, aes(x=PopulationGroup,y=murders$GunMurders)) +
  geom_boxplot(alpha=0.75,fill="plum") +
  coord_flip() +
  theme_bw() +
  scale_y_continuous(breaks = seq(0,1300,200)) +
  annotate("text", x = 4.5, y = 1000, label =
             paste("Range:",round(rbig,2)),size=3) +
  annotate("text", x = 4.2, y = 1000, label =
             paste("Standard Deviation:",round(sbig,0)),size=3) +
  annotate("text", x = 1.5, y = 1000, label =
             paste("Range:",round(rsmall,2)),size=3) +
  annotate("text", x = 1.2, y = 1000, label =
             paste("Standard Deviation:",round(ssmall,0)),size=3)
