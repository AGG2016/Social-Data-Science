Basic Concepts

Variables

An overview of basic concepts is a prerequisite:

  • Datum: The codified expression of a fact.

  • Population: Refers to the collection of attributes, more than to the individuals, cases or observations. Populations can be finite or infinite; the former could be exhaustively measured -with no resources restrictions-, while the latter are conceptual or theoretical: the collection of all decisions of the UN Security Council (“all the decisions have not happended”). In the first case, we deal with “enumerative studies”, while research is “analytical” in the second.

  • Censal Studies: when a finite population is exhaustively meassured.

  • Sample Studies: when due to limitations in resources, the population is meassured only partially.

  • Variable: an attribute which changes from observation to observation.
    • Quantitative, rate or “interval” variables (numeric properties):
      • Continuous
      • Discrete
    • Categorical
      • Nominal (e.g. gender, knowledge field)
      • Ordinal (e.g. Likert Scale, grouped quantitatives -age intervals-)
  • Notation:
    • \(\ n = sample-size, n \epsilon \mathbb{N}\)
    • \(\ m = second-sample-size\)
    • \(\ x = variable\)
    • \(\ x_1,x_2,...,x_n = observations\)
