Correlation

Covariance

\[Cov_(x,y_)=\frac{\sum(x_i-\bar{x})(y_i-\bar{y})}{n-1}\]

Problem: its value depends critically on the units of the measurement; it will be artificially inflated if, for instance, measurements are converted to smaller units.

This must be solved by scaling the the covariance, in terms of the respective standard deviations.

Correlation Coefficient

\[\rho_(x,y_)=\frac{Cov_(x,y_)}{s_x*s_y}\]

Interpretation

  • Moves between -1 and 1 -always-.

  • The higher the absolute value the stronger the correlation.

  • A negative value implies negative correlation.

Thumb rule

  • ρ < 0.3 Weak
  • 0.3 < ρ < 0.7 Moderate
  • 0.7 < ρ Strong

Caveats

  • If x and y are independent, then ρ = 0. However, ρ = 0 does not imply independence, since a strong but non-linear correlation could be present.

  • ¡Quantitative variables only!

Perfect Linear Correlation

ggplot(murders,aes(GunMurders,GunMurders)) +
  geom_point(alpha=0.5,col="red") +
  geom_smooth(method=lm)

Not so obvious example

National Material Capabilities -NMC- version 4.0 1

c14<-subset(nmc,year==1914)
c14c<-cor(c14$milex,c14$pec/1000000,method = "pearson",
    use = "pairwise")
  • ρ(military expenditures,energy consumtion): 0.43
df14<-subset(nmc,year==1914)
df14<-subset(df14,EPI_regions %in% wantedlevels)
  ggplot(df14,aes(x=milex,y=pec/1000000))+
  geom_point(aes(colour = factor(EPI_regions),
                 size=UrbanPopulation/1000000)) +
  geom_smooth(method=lm) +
  geom_text(aes(label=ifelse(milex>250,as.character(stateabb),
                             "")),vjust=2) +
  labs(title="1914",
       caption=paste("Correlation Coefficient:",round(c14c,2))) +
  xlab("Military Expenditures -$US MM 1914-") +
  ylab("Energy Consumption -MM coal-ton equivalents-") +
  scale_y_continuous(labels = comma)
